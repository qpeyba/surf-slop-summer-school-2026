## Примечания по адаптации

1. **Фото** — крупное фото блюда (паста, выпечка и т.д.), размер 360×120, border-radius 16px
2. **«Карта маршрута»** — для «Шеф-стол» вместо карты маршрута — схема проезда или адрес с картой. Можно упростить до статичной картинки с картой + кнопка «Открыть в Яндекс.Картах». На MVP может не быть отдельного экрана.
3. **Кнопка CTA** — sticky bottom, как в Figma: 360×56, padding 24px 16px, border-radius 32px
4. **Теги** — «Новичковый» / «Опытный» + время (~3 ч). Padding чипса: 4px 8px, border-radius: 8px
5. **Информационные блоки** — использовать разделители (1px Light grey) между секциями: Меню, Шеф, Адрес, Дата, Места, Стоимость
6. **Звёзды рейтинга** — минимальная высота touch target 48dp (доступность). Дублируются текстовой меткой: «★ 4.8 (124 оценки)»
