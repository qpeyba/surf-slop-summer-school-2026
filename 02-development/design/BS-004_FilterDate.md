## Версия с активными фильтрами (71-5722)

Отличия от default:
- Чипс «Эта неделя» — Brand/Primary фон, White текст (выбран)
- Поля дат заполнены: «с: 20 июня», «по: 23 июня»
- Чипс «Новичковый» — выбран (Brand/Primary)
- Чипс «Анна» — выбран (Brand/Primary)
- Кнопка «Сбросить» — активна (Black текст)
- Кнопка «Применить» — активна (Brand/Primary фон)

## Примечания по адаптации

1. **«Тип маршрута» → «Уровень сложности»** — для «Шеф-стол»: «Для новичков» / «Для опытных» (вместо «Новичковый» / «Опытный»).
2. **«Инструктор» → «Шеф»** — заменить «Инструктор» на «Шеф» во всех контекстах.
3. **Быстрые фильтры дат** — «Сегодня», «Эта неделя», «Выходные». Для кулинарных классов добавить «Завтра».
4. **Календарь** — по спецификации BS-004 можно добавить компактный месячный календарь между быстрыми фильтрами и диапазоном дат. В Figma-версии календаря нет — только чипсы и поля ввода дат.
5. **«Применить»** — закрывает шторку и обновляет SCR-001. Фильтры сохраняются в состоянии экрана.
6. **Закрытие без сохранения** — свайп вниз или тап по фону → фильтры не применяются.
