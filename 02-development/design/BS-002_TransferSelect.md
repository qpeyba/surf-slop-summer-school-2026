│  │  │  + ручка-индикатор (40×4)                   │││

## Шаг 2: Полноэкранный выбор слота

Повторяет структуру SCR-001 (список карточек), но с фильтрацией:

```
┌─ Статус-бар (54px) ──────────────────────────────┐
│  [← Back]  «Выберите класс» (24px Bold, center)   │
├────────────────────────────────────────────────────┤
│                                                    │
│  Доступные слоты на 7 дней:                        │
│                                                    │
│  ┌─ Карточка класса (w=360, pad: 16px, gap: 12px)┐│
│  │  border-radius: 32px, фон: Light grey           ││
│  │  ... (как в SCR-001, но компактнее)            ││
│  │  ┌─ Кнопка «Записаться» (справа/снизу) ──────┐││
│  │  │  или inline-кнопка                         │││
│  │  └─────────────────────────────────────────────┘││
│  └─────────────────────────────────────────────────┘│
│                                                    │
│  ⓘ Текущая бронь будет отменена автоматически      │
│    при создании новой                               │
│                                                    │
└────────────────────────────────────────────────────┘
```

### Отступы и параметры (шаг 2)

| Элемент | Значение |
|---------|----------|
| Всё как в SCR-001 | те же карточки, отступы, шрифты |
| Заголовок | y=80, 24px Bold, center |
| Back Button | 40×40, круг, x=16, y=74 |
| Инфо-блок внизу | padding: 12px 16px, 14px Regular, grey |

### Что НЕ показывать
- Классы без свободных мест
- Классы с началом ≤ 10 мин
- Отменённые классы
- Тот же слот, на который уже есть бронь

## Шаг 3: Шторка подтверждения переноса

```
┌─ Затемнённый фон ─────────────────────────────────┐
│                                                     │
│  ┌─ Bottom Sheet (как в BS-001) ─────────────────┐│
│  │  ┌─ Заголовок «Подтвердить перенос?» ────────┐││
│  │  └─────────────────────────────────────────────┘││
│  │                                                  ││
│  │  Было:  🍝 Итальянская паста                    ││
│  │         Вт 14.07 · 15:00                        ││
│  │                                                  ││
│  │  Стало: 🥐 Сложная выпечка                      ││
│  │         Сб 18.07 · 13:00                        ││
│  │                                                  ││
│  │  ┌─ Кнопка «Перенести» (w=360, h=56) ─────────┐││
│  │  │  padding: 24px 16px, border-radius: 32px    │││
│  │  └──────────────────────────────────────────────┘││
│  │                                                  ││
│  │  «Отмена» (текст, center)                        ││
│  └──────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────┘
```

### Состояния

| Шаг | Состояние | Поведение |
|-----|-----------|-----------|
| 2 | Loading | Скелетон: 3 карточки-прямоугольника (shimmer) |
| 2 | Empty | «Нет доступных классов для переноса» + «Назад» |
| 3 | Loading | Кнопка → spinner, всё заблокировано |
| 3 | Error (409) | «На выбранном классе больше нет мест» |
| 3 | Error (сеть) | «Не удалось выполнить перенос» |
| 3 | Success | Закрытие + snackbar «Бронь перенесена на 18 июля» |

## Примечания по адаптации

1. **Шаг 2** — список карточек как в SCR-001, но без Tab Bar (это push-экран)
2. **Шаг 1 и 3** — шторки как BS-001: border-radius 32px сверху, ручка-индикатор 40×4, затемнение 0.6
3. **Кнопка «Записаться»** на шаге 2 — inline в карточке (справа) или компактная. Размер: w=120–180, h=40, padding: 6px 12px, border-radius: 32px
4. **Сравнение «Было / Стало»** — две компактные карточки рядом или вертикально
