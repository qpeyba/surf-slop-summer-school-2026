## Примечания по адаптации

1. **Логотип** — заменить сёрф-логотип «Волна» на логотип «Шеф-стол» (шеф-колпак / кулинарная тематика), сохранить размеры 235×40 и позицию x=79, y=88.71
2. **Текст приветствия** — «Добро пожаловать! Войдите, чтобы записаться на кулинарные классы» (вместо «...на прогулку»)
3. **Клавиатура** — оставить стандартную цифровую клавиатуру iOS/Android
4. **Цвет Brand/Primary** — заменить `#00A59D` на тёплый терракотовый
5. **Поток OTP** — 4 отдельных поля по 84×52, border-radius 16px, активное поле с обводкой 2px Brand/Primary, gap между полями 8px
6. **Back Button** — использовать системную стрелку «←» (Material), белый круг 40×40 с тенью
