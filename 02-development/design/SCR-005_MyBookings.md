## Состояния экрана

### Loading
- Скелетон: 3 группы с заголовками-прямоугольниками (shimmer), 1–2 карточки-прямоугольника под каждой

### Empty
- Иллюстрация по центру + «У вас пока нет броней» + кнопка «Перейти к расписанию»

### Error
- «Не удалось загрузить список броней» + кнопка «Повторить»

## Примечания по адаптации

1. **Tabs** — в Figma: «Предстоящие» / «Прошедшие». Для «Шеф-стол» по спецификации SCR-005: секции «Активные», «Завершённые», «Отменённые». Это не табы, а секции с заголовками (более плоская структура). Figma-табы можно адаптировать как быстрое переключение между «Активные» и «История».
2. **Кнопки действий** — в Figma на карточке только статус. По спецификации SCR-005 нужны кнопки: «Отменить» (outline, красный), «Перенести» (outline). Для завершённых: «Оценить шефа» (primary). Добавить эти кнопки в карточку.
3. **Таймер «Через N дней»** — в Figma нет. Добавить для активных броней перед статусом: «Через 2 дня 4 часа» (14px Regular, grey).
4. **Сумма возврата** — для отменённых броней: «К возврату: 1 750 ₽» (14px Regular).
5. **Pull-to-refresh** — поддержать на этом экране.
