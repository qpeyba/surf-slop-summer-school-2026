### Звёзды

| Состояние | Вид |
|-----------|-----|
| Пустая (не выбрана) | ☆ — контур, Secondary grey `#CCCCCC` |
| Заполненная (выбрана) | ★ — заливка, Gold/Warm yellow |
| При наведении/тапе | Заполняются все звёзды слева направо до выбранной |

### Состояния кнопки

| Условие | Состояние |
|---------|-----------|
| 0 звёзд | Disabled (Light grey фон, Grey текст) |
| 1+ звёзд | Active (Brand/Primary фон, White текст) |
| Отправка | Spinner вместо текста, всё заблокировано |

## Вариант Б: Редактирование

Отличается только:
- Заголовок: «Редактировать оценку» (вместо «Оценить шефа»)
- Звёзды: предзаполнены текущей оценкой
- Текстовое поле: заполнено текущим отзывом
- Кнопка: «Сохранить» (вместо «Отправить»)

---

## Состояния

| Состояние | Поведение |
|-----------|-----------|
| Loading | Кнопка → spinner, контент заблокирован |
| Error | «Не удалось сохранить оценку» (14px, Red, под кнопкой) |
| Success | Шторка закрывается, snackbar «Оценка сохранена» |

---

## Примечания по адаптации

1. **Звёзды** — минимальный touch target 48dp для доступности (WCAG 2.1 AA). Дублируются текстовой меткой «N из 5».
2. **Счётчик символов** — 0/500, обновляется при вводе. При превышении — текст красный, ввод блокируется.
3. **Кнопка disabled** — если 0 звёзд: Light grey фон `#F2F2F2`, Grey текст `#797979`.
4. **Отзыв опционален** — можно отправить только звёзды без текста.
