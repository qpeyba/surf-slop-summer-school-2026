# Индекс переиспользуемых логик

> Логики описывают бизнес-правила и алгоритмы, используемые на нескольких экранах.
> Экран ссылается на логику по ID (`LOGIC-XXX`) и не дублирует описание.
> Источник истины для правил — `../2-requirements/` и `../4-design/data-model.md`.

## Правила использования
1. Экран **ссылается** на логику — не дублирует описание правил.
2. Если правило затрагивает только один экран — оно описывается в самом экране, не выносится в логики.
3. Числа (цена, refundAmount, лимиты) — из ответов API, не хардкод.
4. Источник истины — бэкенд; клиентские pre-check — UX-помощники.
