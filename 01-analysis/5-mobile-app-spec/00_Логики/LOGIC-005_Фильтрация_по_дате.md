## Обзор

Фильтрация расписания классов **только по дате** (FR-1.2). Никаких других фильтров (по шефу, уровню сложности, меню) в MVP нет. Диапазон — 7 дней от выбранной даты. Параметры `from` и `to` передаются в `GET /slots`. Выбор даты — через Bottom Sheet BS-004 (календарь / список дат).

### User Story

> Как Клиент, я хочу фильтровать расписание по дате на ближайшие 7 дней,
> чтобы быстро находить классы в удобные для меня дни.

### Бизнес-ценность

- Узкий фокус: только дата — упрощает UI и бэкенд
- 7-дневное окно — мотивирует на скорую запись
- Параметры API (`from`/`to`) строго соответствуют возможностям бэкенда

## Описание логики

### Параметры запроса

```
GET /slots?from={YYYY-MM-DD}&to={YYYY-MM-DD}
```

| Параметр | Тип | Обязательность | По умолчанию |
|----------|-----|----------------|--------------|
| `from` | date | Нет | Сегодня |
| `to` | date | Нет | from + 7 дней |

### Правила
1. Единственный фильтр — **дата** (FR-1.2).
2. Диапазон отображения — **7 дней** от выбранной `from` (FR-1.1).
3. При первом открытии: `from = today()`, `to = today() + 7 days`.
4. При выборе даты в BS-004: `from = выбранная дата`, `to = выбранная дата + 7 days`.
5. Выбранная дата отображается в заголовке / кнопке фильтра на SCR-001.
6. Сброс фильтра (кнопка «Сбросить» / жест) возвращает `from = today()`.

### Формат дат
- API принимает даты в формате `YYYY-MM-DD` (ISO 8601 date).
- Отображение в UI — локализованный формат (зависит от платформы Android).

## API запросы

### GET /slots

**Спецификация:** [`../api/slots/api.yaml`](../api/slots/api.yaml) → `listSlots`

**Параметры:**

| Параметр | Тип | Значение/Источник |
|----------|-----|-------------------|
| `from` | string (date) | Выбранная дата или сегодня |
| `to` | string (date) | from + 7 дней |

**Обработка ответа:**

| Результат | Действие |
|-----------|----------|
| Загрузка | Скелетон-шиммер |
| Успех (200) + данные | Отобразить список классов |
| Успех (200) + пустой массив | Empty state: «Пока нет доступных классов» (FR-1.3) |
| Ошибка 401 | Редирект на SCR-006 (LOGIC-001) |
| Ошибка 5xx | Error state с кнопкой «Обновить» |
| Ошибка сети | Error state с кнопкой «Обновить» |

## Критерии приёмки

| ID | Критерий |
|----|----------|
| AC-001 | **Дано** открыто расписание, **Когда** не выбран фильтр, **Тогда** отображаются классы с сегодня на 7 дней вперёд |
| AC-002 | **Дано** выбрана дата 2026-07-10 в BS-004, **Когда** применён фильтр, **Тогда** запрос `GET /slots?from=2026-07-10&to=2026-07-17` |
| AC-003 | **Дано** выбран фильтр даты, **Когда** нажат «Сбросить», **Тогда** `from` сбрасывается на сегодня |
