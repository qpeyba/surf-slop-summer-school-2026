## Обзор
Bottom Sheet подтверждения отмены брони. Открывается из SCR-005 по кнопке «Отменить» для активной брони. Перед открытием вычисляется `remaining` — время до начала класса — и отображается соответствующее предупреждение: >12ч — бесплатная отмена, ≤12ч — возврат 50%. После подтверждения отправляется `cancelBooking`. Правило — 12 часов (LOGIC-004).

### User Story
> Как Клиент, я хочу отменять свои активные брони прямо в приложении и знать правила удержания стоимости,
> чтобы своевременно освобождать места для других участников.

### Бизнес-ценность
- Прозрачные правила возврата (FR-4.1)
- Предотвращение случайных отмен
- Освобождение мест для других клиентов

## Используемые запросы

### getBooking
**Тип:** REST
**Метод:** GET
**Спецификация:** [../api/bookings/api.yaml](../api/bookings/api.yaml) → `getBooking`

**Триггер:** Инициализация

**Параметры:**
| Параметр | Тип | Обязательность | Источник | Описание |
|----------|-----|----------------|----------|----------|
| `bookingId` | string (UUID) | Да | Входной параметр | ID брони |

**Обработка ответа:**
| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Загрузка | — | Скелетон в шторке |
| Успех (200) | — | Получить `slotId`, перейти к getSlot |
| HTTP 404 | — | Снек «Бронь не найдена», закрыть BS-001 |
| HTTP 401 | — | Редирект на SCR-006 |

### getSlot
**Тип:** REST
**Метод:** GET
**Спецификация:** [../api/slots/api.yaml](../api/slots/api.yaml) → `getSlot`

**Триггер:** После getBooking

**Параметры:**
| Параметр | Тип | Обязательность | Источник | Описание |
|----------|-----|----------------|----------|----------|
| `slotId` | string (UUID) | Да | `booking.slotId` из getBooking | ID слота |

**Обработка ответа:**
| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Загрузка | — | Скелетон в шторке |
| Успех (200) | — | Вычислить `remaining`, показать предупреждение |
| HTTP 404 | — | Снек «Класс не найден», закрыть BS-001 |
| HTTP 401 | — | Редирект на SCR-006 |

### cancelBooking
**Тип:** REST
**Метод:** POST
**Спецификация:** [../api/bookings/api.yaml](../api/bookings/api.yaml) → `cancelBooking`

**Триггер:** Тап «Подтвердить отмену»

**Параметры:**
| Параметр | Тип | Обязательность | Источник | Описание |
|----------|-----|----------------|----------|----------|
| `bookingId` | string (UUID) | Да | Входной параметр | ID брони |

**Обработка ответа:**
| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Загрузка | — | Лоадер на кнопке «Подтвердить отмену» |
| Успех (200) | `refundAmount == null` | Снек «Бронь отменена», закрыть BS-001, обновить SCR-005 |
| Успех (200) | `refundAmount > 0` | Снек «Бронь отменена. К возврату: {refundAmount} ₽», закрыть BS-001 |
| HTTP 409 | `booking_not_active` | Снек «Бронь уже отменена или завершена», закрыть BS-001 |
| HTTP 404 | — | Снек «Бронь не найдена», закрыть BS-001 |
| HTTP 401 | — | Редирект на SCR-006 |
| HTTP 5xx | — | Снек «Произошла ошибка. Попробуйте позже» |
| Сеть | Нет соединения | Снек «Нет соединения. Проверьте подключение» |

## Критерии приёмки

### Позитивные сценарии
| ID | Критерий | Приоритет |
|----|----------|-----------|
| AC-001 | **Дано** `remaining > 12ч`, **Когда** BS-001 открыта, **Тогда** предупреждение «Отмена бесплатна. Подтвердить отмену?» | P0 |
| AC-002 | **Дано** `remaining ≤ 12ч`, **Когда** BS-001 открыта, **Тогда** предупреждение «При отмене возвращается 50% стоимости. Сумма к возврату: {refundAmount} ₽.» | P0 |
| AC-003 | **Дано** нажато «Подтвердить», **Когда** cancelBooking 200 с `refundAmount=null`, **Тогда** снек «Бронь отменена», закрытие BS-001, обновление SCR-005 | P0 |
| AC-004 | **Дано** нажато «Подтвердить», **Когда** cancelBooking 200 с `refundAmount>0`, **Тогда** снек «Бронь отменена. К возврату: {refundAmount} ₽», закрытие BS-001 | P0 |

### Негативные сценарии
| ID | Критерий | Приоритет |
|----|----------|-----------|
| AC-N01 | **Дано** cancelBooking возвращает 409 (booking_not_active), **Когда** ответ получен, **Тогда** снек «Бронь уже отменена или завершена», закрытие BS-001 | P0 |
| AC-N02 | **Дано** cancelBooking возвращает 404, **Когда** ответ получен, **Тогда** снек «Бронь не найдена», закрытие BS-001 | P1 |
| AC-N03 | **Дано** ошибка сети при getBooking/getSlot, **Когда** открытие BS-001, **Тогда** снек с ошибкой, закрытие BS-001 | P1 |
