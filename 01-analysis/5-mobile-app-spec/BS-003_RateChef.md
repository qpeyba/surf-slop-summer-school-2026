## Обзор
Bottom Sheet для выставления или редактирования оценки шефу. Открывается из SCR-005 по кнопке «Оценить шефа» (для завершённой брони без оценки) или «Редактировать оценку» (если оценка уже есть). Upsert-семантика: один endpoint `PUT /bookings/{bookingId}/review` для создания и редактирования. Оценка: 1–5 звёзд + опциональный текст отзыва. Бэкенд пересчитывает рейтинг шефа (`instructorRating` в ответе).

### User Story
> Как Клиент, я хочу оценивать шефа (ставить звёзды и писать отзыв) после завершения класса,
> чтобы делиться обратной связью и влиять на рейтинг шефа.

### Бизнес-ценность
- Обратная связь для студии и других клиентов
- Пересчёт рейтинга шефа (data-model §2.2)
- Редактирование в любой момент

## Используемые запросы

### getBooking
**Тип:** REST
**Метод:** GET
**Спецификация:** [../api/bookings/api.yaml](../api/bookings/api.yaml) → `getBooking`

**Триггер:** Инициализация

**Параметры:**
| Параметр | Тип | Обязательность | Источник | Описание |
|----------|-----|----------------|----------|----------|
| `bookingId` | string (UUID) | Да | Входной параметр | ID брони |

**Обработка ответа:**
| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Загрузка | — | Скелетон в шторке |
| Успех (200) | `status == "Завершена"` | Показать форму: звёзды (текущие или пустые) + поле отзыва |
| Успех (200) | `status != "Завершена"` | Снек «Оценить шефа можно только после завершения класса», закрыть BS-003 |
| HTTP 404 | — | Снек «Бронь не найдена», закрыть |
| HTTP 401 | — | Редирект на SCR-006 |

### upsertReview
**Тип:** REST
**Метод:** PUT
**Спецификация:** [../api/bookings/api.yaml](../api/bookings/api.yaml) → `upsertReview`

**Триггер:** Тап «Отправить» / «Сохранить»

**Body:**
| Параметр | Тип | Обязательность | Источник | Описание |
|----------|-----|----------------|----------|----------|
| `rating` | int (1–5) | Да | Выбор пользователя | Количество звёзд |
| `text` | string | Нет | Ввод пользователя | Текст отзыва |

**Обработка ответа:**
| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Загрузка | — | Лоадер на кнопке |
| Успех (200) | — | Снек «Оценка сохранена», закрыть BS-003, обновить `reviewRating`, `reviewText`, `instructorRating` |
| HTTP 400 | `invalid_rating` | Снек «Рейтинг должен быть от 1 до 5» |
| HTTP 403 | `booking_not_completed` | Снек «Оценить шефа можно только после завершения класса», закрыть |
| HTTP 404 | — | Снек «Бронь не найдена», закрыть |
| HTTP 401 | — | Редирект на SCR-006 |
| HTTP 5xx | — | Снек «Произошла ошибка. Попробуйте позже» |
| Сеть | Нет | Снек «Нет соединения. Проверьте подключение» |

## Элементы экрана

### 1. Сводка брони
| Элемент | Описание | Источник данных |
|---------|----------|-----------------|
| Меню | Название программы | `booking.slot.menu` (getBooking) |
| Дата и время | «12 июля, Вс • 14:00» | `booking.slot.dateTime` |
| Шеф | Имя шефа | `booking.slot.instructor.name` |

### 2. Рейтинг (звёзды)
| Элемент | Описание | Источник данных | Валидация | Действие |
|---------|----------|-----------------|-----------|----------|
| Звёзды 1–5 | Интерактивный выбор | `existingRating` (или 0 для новой) | 1–5 обязательно. Ошибка: «Рейтинг должен быть от 1 до 5» | Тап → выбор |
| Текстовая метка | «{rating} из 5» (accessibility WCAG 2.1 AA) | Выбранный rating | — | — |

### 3. Поле отзыва
| Элемент | Описание | Источник данных | Валидация | Действие |
|---------|----------|-----------------|-----------|----------|
| Поле ввода | Многострочное, placeholder «Ваш отзыв...» | `existingText` (или пусто) | Опционально | Ввод текста |

### 4. Кнопка действия
| Элемент | Описание | Действие |
|---------|----------|----------|
| Кнопка «Отправить» | Primary (новая оценка) | Валидация: `rating` выбран → [upsertReview](#upsertreview) |
| Кнопка «Сохранить» | Primary (редактирование) | Валидация: `rating` выбран → [upsertReview](#upsertreview) |

**Момент валидации:** При отправке формы. Если `rating` не выбран — ошибка под звёздами.

**Условия доступности:**
- Кнопка активна всегда (rating по умолчанию — из существующей оценки или 0; при 0 — показать ошибку валидации)

## Критерии приёмки

### Позитивные сценарии
| ID | Критерий | Приоритет |
|----|----------|-----------|
| AC-001 | **Дано** завершённая бронь без оценки, **Когда** BS-003 открыта, **Тогда** звёзды пустые, поле отзыва пустое, кнопка «Отправить» | P0 |
| AC-002 | **Дано** завершённая бронь с оценкой 4★ и отзывом, **Когда** BS-003 открыта, **Тогда** звёзды = 4, отзыв заполнен, кнопка «Сохранить» | P0 |
| AC-003 | **Дано** выбраны звёзды и отзыв, **Когда** «Отправить», **Тогда** upsertReview 200 → снек «Оценка сохранена», обновление `reviewRating`, `reviewText`, `instructorRating` | P0 |
| AC-004 | **Дано** редактирование: изменены звёзды с 4 на 5, **Когда** «Сохранить», **Тогда** upsertReview 200 → снек «Оценка сохранена», обновление | P0 |

### Негативные сценарии
| ID | Критерий | Приоритет |
|----|----------|-----------|
| AC-N01 | **Дано** getBooking возвращает `status != "Завершена"`, **Когда** BS-003 открыта, **Тогда** снек «Оценить шефа можно только после завершения класса», закрытие | P0 |
| AC-N02 | **Дано** upsertReview 403, **Когда** ответ, **Тогда** снек «Оценить шефа можно только после завершения класса», закрытие | P1 |
| AC-N03 | **Дано** upsertReview 400 (invalid_rating), **Когда** ответ, **Тогда** снек «Рейтинг должен быть от 1 до 5» | P1 |
