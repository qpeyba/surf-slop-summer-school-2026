# SCR-005 — Мои брони

| Поле | Значение |
|------|----------|
| **Название** | Мои брони |
| **Тип** | Основной экран, Tab 2 |
| **FR** | FR-2.4, FR-4.1, FR-4.2, FR-4.4 |
| **UC** | UC-2, UC-3, UC-4, UC-5 |
| **US** | US-7, US-8, US-9, US-10 |

---

## Цель экрана

Показать клиенту все его брони за последние 3 месяца, сгруппированные по статусу. Дать возможность отменить, перенести активную бронь или оценить шефа после завершённого класса.

---

**Что видит пользователь:**

1. **Top App Bar** — заголовок «Мои брони»
2. **Секция «Активные»** — брони в статусе «Активна», отсортированы по дате (ближайшая сверху):
   - Карточка содержит: программа, дата/время, шеф + рейтинг, адрес, тип экипировки, таймер «Через N дней/часов»
   - Кнопки действий: **«Отменить»** (outline, красный) и **«Перенести»** (outline)
3. **Секция «Завершённые»** — брони в статусе «Завершена», за последние 3 месяца:
   - Карточка содержит те же поля
   - Если оценка уже выставлена: показывается количество звёзд клиента + кнопка «Редактировать»
   - Если оценка не выставлена: кнопка «Оценить шефа» (primary)
4. **Секция «Отменённые»** — брони в статусах «Отменена клиентом» / «Отменена студией»:
   - Статус отмены: «Отменена клиентом» или «Отменена студией» + причина (если студией)
   - Сумма возврата (если применимо) — «К возврату: N ₽» (FR-4.1)
   - Без кнопок действий
5. **Pull-to-refresh** — обновление списка броней

## Состояния экрана

### Loading
Скелетон: 3 группы с заголовками-прямоугольниками, под каждой — 1–2 карточки-прямоугольника с shimmer.

### Empty (нет броней за 3 месяца)
Иконка-иллюстрация (пустой билет) + текст «У вас пока нет броней» (UC-5, альт. сценарий 1а). Кнопка «Перейти к расписанию» → SCR-001.

### Empty (есть брони старше 3 месяцев)
Текст «Показаны брони за последние 3 месяца» + секции только с актуальными. Старые брони не показываются (FR-2.4).

### Error
«Не удалось загрузить список броней» + кнопка «Повторить».

### Success
Как в раскадровке. Секции без броней скрываются (не показывать пустой заголовок «Активные (0)»).
