# BS-003 — Оценка шефа

| Поле | Значение |
|------|----------|
| **Название** | Оценка шефа |
| **Тип** | Bottom sheet (из SCR-005) |
| **FR** | FR-4.4 |
| **UC** | UC-4 |
| **US** | US-10 |

---

## Цель

Позволить клиенту выставить или отредактировать оценку шефу: количество звёзд (1–5) и опциональный текстовый отзыв.

**Что видит пользователь:**

1. **Заголовок** — «Оценить шефа» или «Редактировать оценку»
2. **Информация о классе** — программа, имя шефа, дата класса
3. **Звёзды рейтинга** — 5 звёзд:
   - Пустые (☆) до тапа
   - Заполняются золотым цветом (★) слева направо при тапе
   - Можно тапнуть на любую звезду — заполнятся все до неё включительно
   - При редактировании — показывается текущая оценка
   - Минимум 1 звезда для отправки
4. **Поле текстового отзыва** — textarea с placeholder «Поделитесь впечатлениями о шефе (необязательно)». При редактировании — заполнено текущим текстом. Ограничение: до 500 символов (счётчик)
5. **Кнопка «Отправить» / «Сохранить»** — primary, активна только когда выбрана хотя бы 1 звезда
6. **Кнопка «Закрыть»** — текстовая, закрывает без сохранения (если были изменения — подтверждение? Нет, звёзды — быстрый выбор, потери минимальны)

## Состояния

### Loading (отправка)
Кнопка → spinner. Остальное заблокировано.

### Error
Текст ошибки в шторке: «Не удалось сохранить оценку». Кнопка снова активна.

### Success
Шторка закрывается. На SCR-005 обновляется карточка: показывается оценка (звёзды). Snackbar: «Оценка сохранена».

---

## Edge cases

| Ситуация | Поведение |
|----------|-----------|
| Пустой отзыв | Допустимо (отзыв опционален) |
| 0 звёзд | Кнопка неактивна |
| Отзыв > 500 символов | Счётчик символов, блокировка ввода сверх лимита |
| Бронь не в статусе «Завершена» | Оценка недоступна (кнопка «Оценить» скрыта в SCR-005) |

## Навигация

| Откуда | Событие | Куда |
|--------|---------|------|
| SCR-005 | Тап «Оценить шефа» | BS-003 (новая оценка) |
| SCR-005 | Тап «Редактировать» | BS-003 (редактирование) |
| BS-003 | Тап «Отправить» / «Сохранить» (успех) | Закрытие → SCR-005 |
| BS-003 | Тап «Закрыть» / свайп | Закрытие → SCR-005 |
