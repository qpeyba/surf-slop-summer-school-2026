# BS-002 — Перенос брони

| Поле | Значение |
|------|----------|
| **Название** | Перенос брони |
| **Тип** | Bottom sheet → полноэкранный выбор слота (из SCR-005) |
| **FR** | FR-4.2 |
| **UC** | UC-3 |
| **US** | US-9 |

---

## Цель

Позволить клиенту перенести активную бронь на другой доступный слот в одно действие — с одновременной отменой старой брони и созданием новой.

---

**Что видит пользователь:**

**Шаг 1 — Шторка:**
1. Заголовок «Перенести бронь?»
2. Карточка текущей брони (программа, дата, шеф)
3. Кнопка «Выбрать новый слот» (primary)
4. Кнопка «Оставить текущую бронь» (текст, закрывает шторку)

**Шаг 2 — Экран выбора слота:**
1. Top App Bar — «← Назад», заголовок «Выберите класс»
2. Список доступных слотов (аналогично SCR-001, но только с доступными местами)
3. Каждая карточка — компактная: программа, дата/время, шеф + рейтинг, свободные места, стоимость, кнопка «Записаться»
4. Информация внизу: «Текущая бронь будет отменена автоматически при создании новой»
5. **Не показываются:** классы без свободных мест, классы с началом ≤ 10 мин, отменённые классы, тот же слот, на который уже есть бронь

**Шаг 3 — Шторка подтверждения:**
1. Заголовок «Подтвердить перенос?»
2. Сравнение «Было / Стало» — две компактные карточки
3. Кнопка «Перенести» (primary)
4. Кнопка «Отмена» (текст)

## Состояния

### Loading (Шаг 2)
Скелетон: список из 3 карточек-прямоугольников.

### Loading (Шаг 3 — отправка)
Кнопка «Перенести» → spinner, текст «Перенос...». Всё остальное заблокировано.

### Empty (Шаг 2 — нет доступных слотов)
«Нет доступных классов для переноса» + кнопка «Назад».

### Error (Шаг 3 — 409/410 от API)
- 409 (мест нет): «На выбранном классе больше нет мест. Выберите другой слот» (UC-3, искл. 5а)
- 410 (класс начался): «Выбранный класс уже начался. Выберите другой слот»
- Ошибка сети: «Не удалось выполнить перенос. Попробуйте ещё раз»

### Success
Шторка закрывается. На SCR-005 старая бронь → в «Отменённые», новая → в «Активные». Snackbar: «Бронь перенесена на 18 июля».

---

## Edge cases

| Ситуация | Поведение |
|----------|-----------|
| Бронь уже неактивна | «Бронь больше не активна», шторка закрывается, SCR-005 обновляется |
| Клиент выбрал тот же слот | Кнопка «Записаться» не показывается для текущего слота |
| Слот занят в процессе выбора | При тапе «Перенести» — ошибка 409, вернуться к списку |

## Навигация

| Откуда | Событие | Куда |
|--------|---------|------|
| SCR-005 | Тап «Перенести» | BS-002 (шаг 1) |
| BS-002 (шаг 1) | Тап «Выбрать новый слот» | BS-002 (шаг 2, full-screen) |
| BS-002 (шаг 2) | Тап «Записаться» на слоте | BS-002 (шаг 3, шторка подтверждения) |
| BS-002 (шаг 3) | Тап «Перенести» (успех) | Закрытие → SCR-005 |
| BS-002 (любой шаг) | Назад / свайп | Предыдущий шаг / SCR-005 |
