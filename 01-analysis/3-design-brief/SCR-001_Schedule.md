# SCR-001 — Расписание

| Поле | Значение |
|------|----------|
| **Название** | Расписание классов |
| **Тип** | Основной экран, Tab 1 |
| **FR** | FR-1.1, FR-1.2, FR-1.3, FR-4.3 |
| **UC** | UC-1 (шаги 1–2) |
| **US** | US-3 |

---

## Цель экрана

Дать клиенту увидеть доступные кулинарные классы на выбранную дату, оценить их привлекательность по карточке и перейти к деталям для записи.

**Что видит пользователь:**

1. **Top App Bar** — заголовок «Шеф-стол», справа иконка фильтра (воронка) → открывает BS-004_FilterDate
2. **Горизонтальная полоса дат** — скроллируемый ряд из 7 дат, начиная с сегодня. Выбранная дата подсвечена акцентным цветом. Стрелки «← →» по краям для перемотки. Сегодняшняя дата помечена точкой.
3. **Список карточек классов** — вертикальный скролл. Каждая карточка содержит:
   - Фото блюда (слева или сверху)
   - Название программы (жирный шрифт)
   - Звёзды + рейтинг + имя шефа
   - Уровень сложности (бейдж: «Для новичков» / «Для опытных»)
   - Адрес студии
   - Время начала (с длительностью «3 часа»)
   - Свободные места (точное число: «Осталось 3 из 12»)
   - Стоимость
   - Статус (активен — без индикации; отменён — карточка затемнена, перечёркнута, кнопка заблокирована FR-4.3)
4. **Pull-to-refresh** — обновление расписания
5. **Tab Bar** — снизу

### Варианты карточек по статусу

| Статус класса | Вид карточки | Действие по тапу |
|---------------|-------------|------------------|
| Активен, места есть | Полноцветная, кнопка «Записаться» активна | → SCR-002_ClassDetail |
| Активен, мест нет | Полноцветная, кнопка «Мест нет» заблокирована (серый цвет, disabled) | → SCR-002_ClassDetail (посмотреть детали, но нельзя записаться) |
| Активен, до начала ≤ 10 мин | Кнопка записи заблокирована с текстом «Запись закрыта» (FR-3.1) | → SCR-002_ClassDetail (посмотреть, нельзя записаться) |
| Отменён студией | Затемнена (opacity 0.5), перечёркнута, текст «Отменён студией», кнопки нет (FR-4.3) | Неактивна |

## Состояния экрана

### Loading
Скелетон: 3 прямоугольника-карточки с shimmer-эффектом. В каждом: прямоугольник на месте фото, 3 строки текста разной ширины.

### Empty
Иконка-иллюстрация (календарь с грустным лицом) + текст «Пока нет доступных классов» (FR-1.3). Кнопка «Обновить». Если нет классов на выбранную дату, но есть на другие — показывать предложение «Посмотрите классы в другие дни».

### Error
Иконка ошибки + «Не удалось загрузить расписание» + кнопка «Повторить». Если ошибка сети — добавить «Проверьте подключение к интернету».

### Success
Список карточек, как в раскадровке.
