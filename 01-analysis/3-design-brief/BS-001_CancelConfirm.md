# BS-001 — Отмена брони

| Поле | Значение |
|------|----------|
| **Название** | Подтверждение отмены |
| **Тип** | Bottom sheet (из SCR-005) |
| **FR** | FR-4.1 |
| **UC** | UC-2 |
| **US** | US-8 |

---

## Цель

Подтвердить намерение клиента отменить бронь, проинформировать о правилах возврата в зависимости от оставшегося времени, выполнить отмену.

---

```
│  (затемнённый фон — SCR-005)    │
│                                 │
│  ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐  │
│  │                           │  │

│  │                           │  │
│  │  🍝 Итальянская паста     │  │
```

```
│  │  часов. При отмене будет  │  │
│  │  удержано 50% стоимости.  │  │
│  │                           │  │
│  │  Сумма к возврату:        │  │
│  │  1 750 ₽                  │  │
│  │                           │  │
│  │  Возврат будет выполнен   │  │
│  │  владельцем вручную       │  │
│  │                           │  │
│  │  ┌─────────────────────┐  │  │
│  │  │  Да, отменить       │  │  │
│  │  └─────────────────────┘  │  │
│  │                           │  │
│  │  Оставить бронь           │  │
│  │                           │  │
│  └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘  │
│                                 │
└─────────────────────────────────┘
```

**Что видит пользователь:**

1. **Заголовок** — «Отменить бронь?» с иконкой (корзина / крестик)
2. **Краткая информация о брони** — программа, дата, время
3. **Информация о возврате** — зависит от времени до начала:
   - **> 12ч:** зелёная иконка + «До начала более 12 часов — отмена бесплатна»
   - **≤ 12ч:** жёлтая/оранжевая иконка предупреждения + «При отмене будет удержано 50% стоимости» + точная сумма возврата
4. **Примечание о возврате** (для варианта Б) — «Возврат будет выполнен владельцем вручную» (FR-4.1, domain 4.2)
5. **Кнопка «Да, отменить»** — основная, красный цвет (деструктивное действие)
6. **Кнопка «Оставить бронь»** — текстовая, закрывает шторку без действия

---

## Состояния

### Loading (отправка запроса отмены)
Кнопка «Да, отменить» — spinner, текст «Отмена...». Остальной интерфейс шторки затемнён, не интерактивен.

### Success
Шторка закрывается. На SCR-005 появляется snackbar: «Бронь отменена». Карточка брони перемещается в секцию «Отменённые» с анимацией. Если был частичный возврат — показывается сумма к возврату.

### Error
Шторка остаётся открытой. Ошибка в теле шторки: «Не удалось отменить бронь. Попробуйте ещё раз». Кнопка «Да, отменить» снова активна.

---

## Edge cases

| Ситуация | Поведение |
|----------|-----------|
| Бронь уже отменена (повторный тап) | «Бронь уже отменена», шторка закрывается |
| Класс уже начался | Кнопка «Отменить» в SCR-005 скрыта до открытия шторки |
| Нет интернета | Ошибка сети, кнопка возвращается в активное состояние |

## Навигация

| Откуда | Событие | Куда |
|--------|---------|------|
| SCR-005 | Тап «Отменить» | BS-001 |
| BS-001 | Тап «Оставить бронь» | Закрытие → SCR-005 |
| BS-001 | Тап «Да, отменить» (успех) | Закрытие + snackbar → SCR-005 |
| BS-001 | Свайп вниз / тап по фону | Закрытие → SCR-005 |
