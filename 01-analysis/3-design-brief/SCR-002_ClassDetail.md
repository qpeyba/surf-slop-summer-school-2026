# SCR-002 — Карточка класса

| Поле | Значение |
|------|----------|
| **Название** | Карточка класса |
| **Тип** | Экран деталей (push из SCR-001) |
| **FR** | FR-1.4, FR-3.1 |
| **UC** | UC-1 (шаг 2) |
| **US** | US-4, US-5 |

---

## Цель экрана

Дать клиенту полную информацию о выбранном классе, чтобы принять осознанное решение о записи. Показать все атрибуты, фото блюд, рейтинг шефа, стоимость, свободные места.

---

**Что видит пользователь:**

1. **Top App Bar** — стрелка «← Назад», заголовок «Шеф-стол» (или название программы)
2. **Фото блюда** — крупное, на всю ширину экрана, соотношение 2:1. При скролле вниз фото может сжиматься (collapsing toolbar)
3. **Название программы** — крупный заголовок, под ним «от шефа [Имя]»
4. **Бейджи** — уровень сложности и длительность (`CLASS_DURATION` ≈ 3 ч, см. `00-foundations.md` §1) в виде чипсов
5. **Раздел «Меню»** — плоский текст из `Slot.menu` (string), описывающий блюда на классе
6. **Раздел «Шеф»** — аватар (если есть) + имя + рейтинг звёздами + количество оценок в скобках
7. **Раздел «Адрес»** — адрес студии (текст из `Slot.address`). Ссылка «Посмотреть на карте» — заглушка будущего функционала (нет координат в API, в mobile spec не реализуется)
8. **Раздел «Дата и время»** — полная дата + временной интервал + длительность
9. **Раздел «Свободные места»** — прогресс-бар (заполнено / всего) + точное число «Осталось N из M»
10. **Раздел «Стоимость»** — цена + примечание «включая прокат экипировки»
11. **Кнопка CTA «Записаться»** — закреплена снизу (sticky button), во весь экран по ширине

### Варианты кнопки CTA

| Условие | Текст кнопки | Состояние |
|---------|-------------|-----------|
| Места есть, > 10 мин до начала | «Записаться» | Активна (primary color) |
| Мест нет | «Мест нет» | Заблокирована (серый, disabled) |
| ≤ 10 мин до начала | «Запись закрыта» | Заблокирована (FR-3.1) |
| Класс отменён студией | Кнопка скрыта | — |

## Состояния экрана

### Loading
Скелетон: крупный прямоугольник для фото, 3 блока по 3 строки текста, кнопка-прямоугольник снизу.

### Error
«Не удалось загрузить информацию о классе» + кнопка «Повторить». Если класс был отменён в процессе просмотра — обновить статус и показать «Класс отменён студией».

### Success
Как в раскадровке. Все разделы заполнены данными из API.
